udoo-quadcopter

The objective of this project is to control a quadcopter using arduino.

In this project you'll find code to:

Stabilize a quadcopter using accelerometer + gyroscope;
Locate your quadcopter in space using GPS, Barometer and Compass;
Avoid obstacles using a distance sensor.
